# AI 使用复盘

这次主要把 AI 当成辅助工具，而不是直接照搬。

我用 AI 帮忙做了几件事：

1. 梳理数据表关系，确认 session、答案、测评结果、订阅状态应该分开存。
2. 生成一些 mock session，方便测试未付费和已付费两种返回。
3. 帮忙列边界测试，比如非法身高、体重、年龄、目标体重不合理、重复提交、并发版本冲突等。
4. 帮忙检查 README 里是否把接口和测试方式写清楚。

有一次 AI 给的方案我没有采用：它建议未付费接口也返回完整字段，只是把值设成 null。我觉得这样不太好，因为前端仍然能知道被保护数据的结构。最后我改成未付费时直接不返回预测曲线、具体 BMI、热量等字段，只返回预览信息和 lockedFields。
